# Shaw Jiang — Personal Homepage

A personal academic homepage for a cosmology and large-scale structure researcher at the University of Chicago.

## Features

- Dark cosmic theme with animated star-field background
- Glassmorphism-styled cards
- Responsive layout (desktop & mobile)
- Smooth scroll navigation
- Fade-in animations on scroll

## Usage

Open `index.html` in any modern browser — no build step required.

## Structure

```
index.html   — Main page
style.css    — Stylesheet
script.js    — Animations & interactivity
```
